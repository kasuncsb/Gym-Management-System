# PowerWorld Gyms - Frontend

Modern, responsive web application for PowerWorld Gyms Management System.

## Tech Stack

- **Framework**: Next.js 15+ (App Router)
- **Styling**: Vanilla CSS (per requirements)
- **Language**: TypeScript  
- **API Client**: Axios
- **State**: React Context API

## Features

✨ Modern glassmorphism design  
✨ Vibrant color palette (Cyan + Orange)  
✨ Smooth animations & micro-interactions  
✨ Fully responsive (mobile/tablet/desktop)  
✨ Role-based routing (Member/Trainer/Manager/Admin)  
✨ QR code display & scanning  
✨ Real-time subscription validation  
✨ Dark mode support  

## Setup

```bash
cd frontend
npm install
npm run dev
```

App runs on `http://localhost:3000`
